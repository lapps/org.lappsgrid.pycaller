org.lappsgrid.pycaller
======================

Python Function Caller from Java

### Usage

```xml
        <dependency>
            <groupId>org.lappsgrid</groupId>
            <artifactId>pycaller</artifactId>
            <version>0.0.4</version>
        </dependency>
```

### Example

```java
    PyCaller.call(pythonPath, methodName, param1, param2, ...);
```
